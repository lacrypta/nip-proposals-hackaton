## 1. Capability Description

The `Date` capability makes the `Date.Date()` constructor and the `Date.now()` static method available.

## 2. Available Symbols

The following table details what symbols are made available by this capability:

| Symbol Name                  |
| ---------------------------- |
| `Date.Date()`                |
| `Date.now()`                 |
